# 铁腕 - 设计规范文档

## 1. 品牌概述

**产品名称**: 铁腕  
**产品定位**: 智能任务记忆与安排助手  
**核心功能**: 基于艾森豪威尔矩阵的任务优先级管理  
**品牌理念**: 掌控每一刻，让重要的事不再被紧急的事淹没

## 3. 字体系统

### 3.1 字体家族

```css
font-family: "Aeonik Mono", sans-serif;
```

**备用字体**: 系统无衬线字体栈

### 3.2 字体层级

| 层级 | 大小 | 字重 | 字间距 | 用途 |
|------|------|------|--------|------|
| H1 标题 | `clamp(40px, 5.5vw, 58px)` | 400 | 8px | 主标题 |
| H2 副标题 | 15px | 400 | - | 描述文字 |
| 导航链接 | 12px | 500 | 0.6px | 导航菜单 |
| 按钮文字 | 13px | 700 | 0.8px | 按钮 |
| 卡片标题 | 11px | 700 | 0.8px | 象限标题 |
| 任务文字 | 11.5px | 400 | - | 任务列表项 |
| 标签文字 | 10px | 700 | 0.5px | 徽章、标签 |

### 3.3 文字样式规范

- **大写转换**: 按钮、标签、导航使用 `text-transform: uppercase`
- **行高**: 正文 1.7，标题 1.15

## 10. 响应式断点

| 断点 | 宽度 | 调整 |
|------|------|------|
| 桌面 | > 1024px | 完整布局 |
| 平板 | ≤ 1024px | 隐藏浮动装饰，调整间距 |
| 移动 | ≤ 640px | 单列布局，缩小字体 |

---

## 11. 设计原则

1. **清晰优先**: 任务优先级通过颜色和图标清晰区分
2. **简洁高效**: 减少文字，使用图标传达信息
3. **温暖专业**: 暖米色背景营造亲和感，深色窗口体现专业
4. **即时反馈**: 所有交互元素都有明确的悬停状态
5. **视觉层次**: 通过颜色、大小、间距建立清晰的信息层级
